# ローカル環境構築手順

## ビルド
```
docker-compose build
```

## node_modulesインストール

コンテナ一時起動
```
docker-compose run --rm node sh
```

コンテナ内でnode_modulesインストール
```
yarn install
```

終わったらコンテナ終了
```
exit
```

## コンテナ起動
```
docker-compose up -d
```

## 画面起動
http://localhost:3000
